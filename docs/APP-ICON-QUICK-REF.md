# App Icon Quick Reference

## All Required Files

| Filename | Dimensions | Purpose |
|----------|------------|---------|
| `Icon-1024.png` | 1024×1024 | App Store |
| `Icon-180.png` | 180×180 | iPhone App (3x) |
| `Icon-120.png` | 120×120 | iPhone App (2x) & Spotlight (3x) |
| `Icon-87.png` | 87×87 | iPhone Settings (3x) |
| `Icon-80.png` | 80×80 | iPhone Spotlight (2x) |
| `Icon-60.png` | 60×60 | iPhone Notification (3x) |
| `Icon-58.png` | 58×58 | iPhone Settings (2x) |
| `Icon-40.png` | 40×40 | iPhone Notification (2x) |

## Must-Have Checklist

- [ ] All files are PNG format
- [ ] No transparency (solid background)
- [ ] No rounded corners
- [ ] Square aspect ratio (1:1)
- [ ] High contrast, simple design
- [ ] Recognizable at 40×40 pixels
- [ ] RGB color space
- [ ] No text (graphical only)

## Color Recommendations

**Primary Gradient:**
```
Start: #007AFF (Blue)
End:   #5856D6 (Purple)
Angle: 135° diagonal
```

**Background:**
```
White:  #FFFFFF
or
Light:  #F0F0F5
```

## Design Concepts

### Option 1: "K" Letter + Dumbbell
- Bold "K" with dumbbell integrated
- Blue → Purple gradient
- Modern, branded

### Option 2: Dumbbell Icon ⭐ Recommended
- Minimalist dumbbell
- Circular background
- Clean, professional

### Option 3: Progress Ring
- Circular progress indicator
- Athletic theme
- Dynamic, motivational

### Option 4: AI + Fitness
- Dumbbell + circuit pattern
- Tech-forward
- Shows "smart fitness"

## Figma Quick Start

1. Go to https://figma.com
2. Create frame: 1024×1024
3. Design your icon
4. Duplicate and resize to other sizes
5. Export all as PNG

**Time Required:** 1-2 hours
